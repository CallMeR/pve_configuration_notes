# Proxmox VE 折腾手记

## 介绍
PVE 虚拟化平台的安装以及折腾手记。  

- PVE ISO 版本：8.1-1 (更新时间: 2023-11-23)

- 演示机：
    - CPU：英特尔奔腾 Silver N6005 处理器
    - 内存：16 GB
    - 网卡：英特尔以太网控制器 I226-V
    - 硬盘：500 GB NVMe 固态硬盘

- PVE 网络：
    - IPv4 网络
        - IP 地址：`172.16.1.254`
        - 子网掩码：`255.255.255.0` ( 即 `/24` )
        - 网关：`172.16.1.1`
        - DNS：`172.16.1.1`
    - IPv6 网络
        - 首选 `SLAAC` 自动配置
        - IPv6 ULA 网络使用 `fdac::/64` 作为演示
